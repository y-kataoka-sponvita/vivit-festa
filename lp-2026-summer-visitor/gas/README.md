# クーポン登録フォーム × スプレッドシート連携（GAS）

来場者向けLPのメール登録を、Googleスプレッドシートに保存し、登録者へ自動返信メールを送るための Google Apps Script です。

- スクリプト本体：[`Code.gs`](./Code.gs)
- フロント側：`../index.html` の `FORM_ENDPOINT` にウェブアプリURLを貼ると実送信されます（空のままだと送信せず完了画面のみのプレビュー動作）。

---

## セットアップ手順

### 1. スプレッドシートを用意
1. 保存先にしたい Google スプレッドシートを新規作成（例：「Vivitフェスタ クーポン登録」）。
2. メニューの **拡張機能 → Apps Script** を開く。

### 2. スクリプトを貼り付け
1. 開いた Apps Script エディタの `コード.gs`（既定ファイル）の中身を全消しし、[`Code.gs`](./Code.gs) の内容を丸ごと貼り付けて保存。
2. 先頭の設定値は必要に応じて変更：
   - `SHEET_NAME` … 書き込み先シート名（なければ自動作成。ヘッダー行も自動で付きます）
   - `SEND_CONFIRMATION` … 登録直後の自動返信メールを送るなら `true`、不要なら `false`
   - `MAX_REGISTRATIONS` … 先着枠（既定100）。超過分は記録されますがレスポンスに `full: true` が付きます

### 3. ウェブアプリとしてデプロイ
1. 右上 **デプロイ → 新しいデプロイ**。
2. 種類（歯車）で **ウェブアプリ** を選択。
3. 設定：
   - 実行するユーザー：**自分**
   - アクセスできるユーザー：**全員**（＝ログイン不要。LPから匿名で送信するため必須）
4. **デプロイ** を押し、初回は権限承認（自分のGoogleアカウントで許可）。
5. 表示される **ウェブアプリURL**（`https://script.google.com/macros/s/XXXXXXXX/exec`）をコピー。

### 4. LPに接続
`../index.html` の `<script>` 内の以下を、コピーしたURLに差し替える：

```js
var FORM_ENDPOINT = 'https://script.google.com/macros/s/XXXXXXXX/exec';
```

> このURLを貼ってもらえれば、こちらで差し替えコミットもできます。

---

## 動作確認
- ブラウザでウェブアプリURLをそのまま開く → `{"ok":true,"message":"..."}` が出れば稼働中。
- LPのフォームからテスト送信 → スプレッドシートに1行増え、登録したアドレスに自動返信メールが届けばOK。

## スプレッドシートの列
| 登録日時 | メールアドレス | お名前 | 流入元 | 受付番号 |
|---|---|---|---|---|

受付番号は登録順の連番。**1〜100 が先着100名枠**の目安になります。

---

## 仕様・運用メモ
- **重複登録**：同じメールアドレスは2回目以降スキップ（記録済み扱いで `duplicated: true` を返す）。
- **クーポン本体は送りません**：自動返信は「登録ありがとう」の確認のみ。300円クーポンは**開催の約1週間前**に別途一斉送信する運用（LPのFAQ・完了画面の文言と一致）。一斉送信もGASで組めます（必要なら別途）。
- **メール送信量の上限**：無料のGmailアカウントは `MailApp` で **1日100通**まで、Google Workspace は1日1500通まで。先着100名なら無料枠内。上限に達すると送信のみ失敗しますが、**スプレッドシートへの記録は成功**します。
- **CORS**：GASウェブアプリはCORSヘッダーを返さないため、フロントは `fetch(..., { mode: 'no-cors' })` で送信し、レスポンスは読まずに完了画面へ遷移します（記録優先の fire-and-forget）。
- **コードを直したら再デプロイ**：`Code.gs` を編集したら **デプロイ → デプロイを管理 → 編集（鉛筆）→ バージョン：新バージョン → デプロイ** で反映。URLは変わりません。
